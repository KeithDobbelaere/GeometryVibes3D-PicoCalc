# PicoCalc SD / FAT32 Drivers (Third-Party)

This directory contains SD card + FAT32 drivers copied from:

- **picocalc-text-starter** by **Blair Leduc**  
  Repository: https://github.com/BlairLeduc/picocalc-text-starter  
  License: **MIT**

Files included (with minimal integration changes as needed for this project):
- `sdcard.c`, `sdcard.h`
- `fat32.c`, `fat32.h`
- `southbridge.c`, `southbridge.h`
- `clib.c` (newlib stdio bindings)
